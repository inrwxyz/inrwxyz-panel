What is this?
----------------------------------------
One Piece Panel is a local, self-hosted
Minecraft server control panel.

• Runs only on your PC
• No cloud
• No tracking
• No system-level install
• Everything stays inside this folder

----------------------------------------
First Time Setup (ONLY ONCE)
----------------------------------------
1) Extract the ZIP fully
2) Open the extracted folder
3) Double-click: install.bat
4) Wait until installation completes
5) Installer will disable itself automatically

DO NOT run install.bat again.

----------------------------------------
Daily Use
----------------------------------------
1) Double-click: start-panel.bat
2) Open browser
3) Go to:
   http://localhost:5000/panel.html
4) Login using the credentials above

----------------------------------------
Creating a Server
----------------------------------------
• Use the panel to create servers
• Set server name and RAM
• Start / Stop / Delete from panel
• Server files are created inside:
  servers/<your-server-name>/

----------------------------------------
File Manager
----------------------------------------
• Edit config files (.yml, .properties, etc.)
• Upload plugins
• Upload worlds (world, world_nether, world_the_end)
• Delete files/folders if needed

WARNING:
Wrong edits can break servers.
Always keep backups of important worlds.

----------------------------------------
Security Notice
----------------------------------------
• Use only on:
  - localhost
  - private LAN
  - private VPN (Radmin / Hamachi)

• DO NOT expose this panel to public internet
• Change default username/password
• Share access only with trusted people

----------------------------------------
System Safety
----------------------------------------
• No Windows registry changes
• No global PATH changes
• No background services installed
• Java and Node are portable (folder-based)

To remove everything:
→ Just delete this folder.
